# TOPページ 画像生成プロンプト（方向性確認用）

> 目的: 検討で固めた方向性（[TOPページ_デザイン検討.md](./TOPページ_デザイン検討.md) の 7〜8 章）を
> 画像生成AIで静止画にし、**ビジュアルで方向性を判断する**ためのプロンプト集。

## 使い方・注意

- 画像生成AI（Midjourney / DALL·E / Imagen / Stable Diffusion など）に貼り付けて使う。
- **日本語のUIテキストは正確に描けない**ことがほとんど。文字の正しさは気にせず、
  **配色・モチーフ・質感・余白・全体の空気感**を見て判断する。
- まず「メインプロンプト」で1枚出し、しっくり来なければ「バリエーション差し替え」で振る。
- 縦構図（スマホ）指定。モデルによっては末尾の比率指定（`--ar 9:16` 等）を各ツールの記法に合わせる。

---

## メインプロンプト（コピペ用）

```
スマートフォン縦画面の学習クイズアプリ「ヒューリスティクス10本ノック」のトップ画面のUIデザイン。
テーマは「エディトリアル × リソグラフ印刷のようなざらついた質感 × モダンな幾何学」。
知的さ6割・遊び4割のバランス。

配色:
- 主役は深いインディゴ（#4F46E5）＝賢さを担う色
- 背景は寒色寄りのライトグレー（#F4F5F7）でクリーンに
- 差し色として、くすんだ暖色（テラコッタ／マスタード系）をごく控えめに使用

モチーフ:
- ニールセンのユーザビリティ10原則を象徴する10個の幾何学シェイプ
  （円・三角形・半円・弧・正方形・点グリッド・直線などのシンプルな図形）を、
  フラット塗り＋粒状のグレイン（ノイズ）テクスチャで描く
- それらを画面の余白に構成的・リズミカルに配置し、ゆるやかな情景のように見せる

トーン:
- オライリー書籍の表紙のような、知性と親しみが同居した雰囲気
- chillで落ち着いた、ポップになりすぎない印象（学問的）
- クリーンでエディトリアルなサンセリフのタイポグラフィ
- 立体的な光沢・強いドロップシャドウ・グラスモーフィズムは使わない（フラット寄り）

レイアウト:
- 縦1カラム、中央〜上部に大きめの見出し、その下に短い説明文
- 下部に2つの角丸ボタン（メインはインディゴ塗り、サブはアウトライン）
- 全体に上質な余白

印刷物のような上質なフラットデザイン。9:16の縦構図。
```

| プロンプトの記述 | 由来（検討メモ） |

| エディトリアル × リソ × 幾何学 | デザイン言語（7章） |
| 知的さ6:遊び4 | Q1 |

| くすんだ暖色を控えめに | Q13 |

| 縦1カラム・スマホ | Q2 |

## バリエーション差し替え

メインプロンプトの該当箇所を、以下に差し替えて振り比べる。

### 暖色アクセント（Q13・色味おまかせ）
- A: `くすんだテラコッタ（#C2693F 程度）`
- B: `マスタード／辛子色（#C9962B 程度）`
- C: `スモーキーなコーラル（#D17A6A 程度）`
- D: `差し色なし（インディゴ＋無彩色のみ）` ← 賢さ最大・最も抑制的

### 幾何学シェイプの配置
- 散りばめ: `図形を余白にランダムにリズミカルに散らす`
- グリッド: `10個の図形を整然としたグリッド状に並べる`
- ヒーロー: `数個の大きな図形で画面上部に1つの構成（ゆるい情景）をつくる`

### グレインの強さ
- 弱: `ごく薄いグレイン`
- 中: `はっきりとした紙のようなグレイン`
- 強: `粗いリソグラフ印刷のような強いグレインと版ずれ風の質感`

### 全体トーンの振れ幅
- 知的寄り: `ミニマルで端正、図形は少なめ`
- 遊び寄り: `図形を多めに、暖色をやや効かせて軽やかに`

## ネガティブ（避けたい要素）

> ネガティブプロンプト対応モデルでは下記を指定。非対応モデルでは本文に「〜は避ける」と書く。

```
ポップで子供っぽい配色, ネオンカラー, 3Dレンダリング, 写真・実写, 過度なグラデーション,
ガラス質感・グラスモーフィズム, 強い光沢や立体的なドロップシャドウ, ごちゃごちゃした装飾,
絵文字, ステレオタイプなテック感（回路・HUD）, 安っぽいストックイラスト感
```
